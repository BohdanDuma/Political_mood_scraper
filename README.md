# YouTube Comments Sentiment Scraper & Analyzer (POLIT_SCARPER)

Інструмент для автоматичного збору коментарів з YouTube та їхнього аналізу тональності (sentiment analysis) за допомогою нейромережевих моделей. Проєкт контейнеризований через Docker і підготовлений для тривалої роботи на сервері.

---

## 📌 Коротко

- Збирає коментарі з відео/каналів YouTube.
- Аналізує тональність коментарів (позитивна / негативна / нейтральна) з використанням `cardiffnlp/twitter-xlm-roberta-base-sentiment`.
- Зберігає результати у локальну SQLite базу для подальшої аналітики.
- Повністю працює в Docker-контейнері; ваги моделей кешуються локально.

---

## 🚀 Можливості

- Автоматичний скрапінг коментарів з YouTube.
- Нейромережевий аналіз тональності (Hugging Face Transformers).
- Локальна база даних SQLite для результатів та метаданих.
- Кеш моделей у `cache/huggingface` — не потребує повторного завантаження при рестарті.
- Можливість запуску періодичних завдань (наприклад, через Cron у хості).

---

---

## 🔧 Початок роботи (Quickstart)

1) Скопіюйте `.env.example` у `.env` та заповніть необхідні змінні оточення (API-ключі тощо).

2) Побудуйте та запустіть контейнери Docker:

```bash
docker compose up --build -d
```

3) Запустіть збір та аналіз вручну (всередині контейнера):

```bash
docker compose exec yt_scraper python -m src.main
```

4) Стежте за логами:

```bash
docker compose logs -f yt_scraper
```

## ⚙️ Конфігурація

- Використовуйте файл `.env` для зберігання секретів та налаштувань (не додавайте `.env` у git).
- Модель та її вага кешуються у `cache/huggingface` (директорія ігнорується в `.gitignore`).
- База даних зберігається в `data/` (також ігнорується в репозиторії).

---

## 📂 Структура проекту

```text
POLIT_SCARPER/
├── cache/                  # Локальний кеш ваг (ігнорується в Git)
│   └── huggingface/hub/
├── data/                   # Локальна база даних та експортні файли
├── src/                    # Код додатку
│   ├── main.py             # Точка входу
│   ├── DataTransform.py    # Обробка даних та NLP пайплайн
│   ├── YouTubeLoader.py    # Логіка скрапінгу YouTube
│   └── DatabaseConnector.py# Збереження в SQLite
├── .env                    # Локальні секрети (не в репо)
├── Dockerfile
├── docker-compose.yml
├── pyproject.toml
└── requirements.txt
```

---

## Збережені дані та кеш

- `cache/huggingface` — кеш ваг моделі, щоб уникнути повторних завантажень (великий розмір ~1.1 GB).
- `data/youtube_analytics.db` — основна SQLite база з результатами аналізу.
